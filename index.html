Project description

Trial.
ImageJ is a public domain Java image processing program extensively used in life sciences. Active Segmentation plugin is the redesign of existing Trainable Weka Segmentation (TWS) of ImageJ. The main functionality of the platform was already developed in the context of GSOC 2016 but the platform is still under development. The Active Segmentation provides generic functionality and user friendly interface so that the user can include the state of the art filters and machine learning frameworks from the WEKA library:

  • Active learning.
  • Multi-instance learning designed by third party in a robust manner
